# 创建快捷方式文件，扩展名为desktop，内容如下(简化版)：

```
[Desktop Entry]
Name=
Comment=
Exec=
Icon=
Terminal=
Type=Application
StartupNotify=true
```

## 解释：

1. Name（名称）
2. Comment（程序注释）
3. Exec（执行路径，有些需要加 sh，就是打开方式）
4. Icon（图标文件位置，带扩展名）
5. Terminal（是否在终端显示，一般为false）
6. Type（类型，默认是Application）
7. StartupNotify（启动通知）
